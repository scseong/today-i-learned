## 괄호 문자 제거 (스택)

### 1. 문제 정보

- **문제 제목**: 괄호 문자 제거 (스택)
- **카테고리**: 자료구조, 스택(Stack)

### 2. 문제 설명

- **핵심 목표**: 소괄호 `(`와 `)` 사이에 있는 모든 문자(다른 괄호 포함)를 제거하고 괄호 밖에 있는 문자들만 추출한다.
- **제한 사항**:
  - 문자열 길이 <= 100
- **입출력 예시**:
  - Input: (A(BC)D)EF(G(H)(IJ)K)LM(N)
  - Output: EFLM

### 3. 접근 방식 및 알고리즘

- **사용한 알고리즘**:
  1. 스택(Stack) 자료구조
- **풀이 로직**:
  - 문자열을 하나씩 순회하며 모든 문자(`(` 및 일반 문자)를 스택에 push한다.
  - 만약 현재 문자가 닫는 괄호 `)`라면 스택에서 pop, 그 결과값이 여는 괄호 `(`일 때까지 반복
  - 모든 순환이 끝난 뒤 스택에 남아있는 문자들을 join하여 반환

### 4. 성능 측정 결과 (Performance)

| **구분**       | **실행 시간 (평균)** | **시간 복잡도** | **공간 복잡도** |
| -------------- | -------------------- | --------------- | --------------- |
| **Solution 1** | **0.0673ms**         | O(N)            | O(N)            |

## 💻 JavaScript 풀이 코드

Solution 1: 스택과 while 루프를 활용한 문자 추출

```js
function solution1(s) {
  let stack = [];

  for (const x of s) {
    if (x === ")") {
      while (stack.pop() !== "(");
    } else {
      stack.push(x);
    }
  }

  return stack.join("");
}
```
