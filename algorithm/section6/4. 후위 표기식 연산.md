## 후위 표기식 연산 (Postfix)

### 1. 문제 정보

- **문제 제목**: 후위 표기식 연산 (Postfix)
- **카테고리**: 자료구조, 스택(Stack)

### 2. 문제 설명

- **핵심 목표**: 후위 표기식(피연산자 뒤에 연산자가 오는 식)이 주어지면 스택을 이용해 계산 결과를 도출한다.
- **입출력 예시**:
  - Input: 352+\*9-
  - Output: 12

### 3. 접근 방식 및 알고리즘

- **사용한 알고리즘**:
  1. 스택(Stack) 자료구조
- **풀이 로직**:
  - 문자열을 순회하며 isNaN()으로 숫자와 연산자를 구분
  - 숫자일 때: Number()로 형변환 후 스택에 push
  - 연산자일 때 pop()을 두 번 실행하여 피연산자 두 개를 추출하여 연산
  - 계산된 결과값을 다시 스택에 push

### 4. 성능 측정 결과 (Performance)

| **구분**       | **실행 시간 (평균)** | **시간 복잡도** | **공간 복잡도** |
| -------------- | -------------------- | --------------- | --------------- |
| **Solution 1** | **0.0594ms**         | O(N)            | O(N)            |

## 💻 JavaScript 풀이 코드

Solution 1: 스택을 활용한 후위 표기식 연산 구현

```js
function solution1(s) {
  let stack = [];

  for (let x of s) {
    if (isNaN(x)) {
      const b = stack.pop();
      const a = stack.pop();

      if (x === "+") stack.push(a + b);
      else if (x === "*") stack.push(a * b);
      else if (x === "-") stack.push(a - b);
      else if (x === "/") stack.push(Math.floor(a / b));
    } else stack.push(Number(x));
  }

  return stack[0];
}
```
