## 올바른 괄호

### 1. 문제 정보

- **문제 제목**: 올바른 괄호
- **카테고리**: 자료구조, 스택(Stack)

### 2. 문제 설명

- **핵심 목표**: 레이저와 쇠막대기의 배치(괄호)가 주어질 때, 절단된 쇠막대기 조각의 총 개수를 구한다.
- **판별 규칙**:
  - `()` (인접한 괄호): 레이저
  - `(` (여는 괄호): 쇠막대기의 시작
  - `)` (인접하지 않은 닫는 괄호): 쇠막대기의 끝
- **입출력 예시**:
  - Input: `()(((()())(())()))(())`
  - Output: 17

### 3. 접근 방식 및 알고리즘

- **사용한 알고리즘**:
  1. 스택(Stack) 자료구조
  2. 카운팅
- **풀이 로직**:
  1. 스택(Stack)
  - 괄호를 실제로 스택에 넣고 빼며 구조를 파악
  - `s[i-1]`을 통해 직전 상태를 확인하여 레이저와 막대 끝을 구분
  2. 카운팅
  - count 변수로 현재 겹쳐진 막대 수를 관리
  - 레이저임을 확인하면 즉시 count를 더하고 인덱스를 건너뜀
  -

## 💻 JavaScript 풀이 코드

Solution 1: Stack

```js
function solution1(s) {
  let answer = 0;
  let stack = [];

  for (let i = 0; i < s.length; i++) {
    if (s[i] === "(") stack.push("(");
    else {
      stack.pop();
      if (s[i - 1] === "(") answer += stack.length;
      else answer++;
    }
  }
  return answer;
}
```

Solution 2: Counting

```js
function solution2(s) {
  let count = 0;
  let answer = 0;

  for (let i = 0; i < s.length; i++) {
    if (s[i] === "(") {
      if (s[i + 1] === ")") {
        answer += count;
        i++;
      } else count++;
    } else {
      count--;
      answer += 1;
    }
  }
  return answer;
}
```
