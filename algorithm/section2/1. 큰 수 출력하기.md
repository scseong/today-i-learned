## 1. 큰 수 출력하기

N(1<=N<=100)개의 정수를 입력받아, 자신의 바로 앞 수보다 큰 수만 출력하는 프로그램을 작성
(첫 번째 수는 무조건 출력한다)

풀이:

1. 명령형 프로그래밍

```js
function solution(arr) {
  const answer = [arr[0]];

  for (let i = 1; i < arr.length; i++) {
    if (arr[i] > arr[i - 1]) {
      answer.push(arr[i]);
    }
  }

  return answer;
}
```

2.  선언형 프로그래밍 - `filter()` 메서드 사용

```js
function solution(arr) {
  return arr.filter((num, idx) => {
    if (idx === 0) return true;
    return num > arr[idx - 1];
  });
}
```
