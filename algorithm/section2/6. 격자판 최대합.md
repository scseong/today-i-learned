## 6. 격자판 최대합

5\*5 격자판에 아래와 같이 숫자가 적혀있습니다.

10 13 10 12 15
12 39 30 23 11
11 25 50 53 15
19 27 29 37 27
19 13 30 13 19

N\*N의 격자판이 주어지면 각 행의 합, 각 열의 합, 두 대각선의 합 중 가 장 큰 합을 출력합니다.

풀이:

```js
/*
행 합:            index 0 -> length | x = 0, y = 0 -> length
열 합:            index 0 -> length | x = 0 -> length, y = 0
대각선 합:        index 0 -> length | x = y -> length
대각선 합 반대:    index 0 -> length | x = 0 -> length, y = length - index 
*/

function solution1(arr) {
  let answer = Number.MIN_SAFE_INTEGER;
  let limit = arr.length;
  let [xSum, ySum, dSum, rDSum] = [0, 0, 0, 0];

  for (let i = 0; i < limit; i++) {
    xSum = ySum = 0;
    for (let j = 0; j < limit; j++) {
      xSum += arr[i][j];
      ySum += arr[j][i];

      if (i === j) dSum += arr[i][j];
      if (j === limit - 1 - i) rDSum += arr[i][j];
    }

    answer = Math.max(answer, xSum, ySum);
  }

  answer = Math.max(answer, dSum, rDSum);
  return answer;
}
```
