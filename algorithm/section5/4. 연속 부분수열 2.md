## 4. 연속 부분수열 2

### 1. 문제 정보

- **문제 제목**: 4. 연속 부분수열 2
- **카테고리**: 효율성, 두 포인터

### 2. 문제 설명

- **핵심 목표**: : 연속된 부분수열의 합이 $M$ 이하가 되는 모든 경우의 수를 구한다.
- **제한 사항**:
  - N <= 100,000, M <= 100,000,000
- **입출력 예시**:
  - Input: M=5, [1, 3, 1, 2, 3]
  - Output: 10

### 3. 접근 방식 및 알고리즘

- **사용한 알고리즘**:
  1. 두 포인터
- **풀이 로직**:
  1. rt 포인터가 오른쪽으로 한 칸씩 이동하며 sum에 현재 값을 더한다
  2. 만약 sum이 M을 초과하면, sum이 M 이하가 될 때까지 lt 포인터를 오른쪽으로 이동시키며 sum에서 값을 뺀다
  3. sum <= M이 된 순간, 현재 rt를 끝점으로 하는 부분수열 중 합이 M 이하인 것의 개수는 rt - lt + 1이다. 이를 answer에 누적한다.

### 4. 성능 측정 결과 (Performance)

| **구분**       | **실행 시간 (평균)** | **시간 복잡도** | **공간 복잡도** |
| -------------- | -------------------- | --------------- | --------------- |
| **Solution 1** | **0.069ms**          | O(N)            | O(1)            |

## 💻 JavaScript 풀이 코드

Solution 1: rt를 이동하며 합을 누적하고 m을 초과할 경우 lt를 이동, 각 단계에서 rt를 포함한 가능한 부분수열 개수(rt - lt + 1)를 합산

```js
function solution(m, arr) {
  let answer = 0;
  let sum = 0;
  let lt = 0;

  for (let rt = 0; rt < arr.length; rt++) {
    sum += arr[rt];

    while (sum > m) {
      sum -= arr[lt++];
    }

    answer += rt - lt + 1;
  }

  return answer;
}
```
