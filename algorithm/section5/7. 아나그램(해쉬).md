## 아나그램(해쉬)

### 1. 문제 정보

- **문제 제목**: 아나그램(해쉬)
- **카테고리**: 효율성, 해쉬 (Hash)

### 2. 문제 설명

- **핵심 목표**: 길이가 같은 두 문자열의 알파벳 구성(종류와 개수)이 일치하는지 판별한다.
- **제한 사항**:
  - 단어의 길이 N ≤ 100
  - 대소문자 구분
- **입출력 예시**:
  - Input: AbaAeCe, baeeACA
  - Output: YES

### 3. 접근 방식 및 알고리즘

- **사용한 알고리즘**:
  1. 해쉬 맵
  2. 정렬
- **풀이 로직**:
  1. Solution 1
  - 첫 번째 문자열을 순회하며 각 문자의 빈도수를 Map에 저장
  - 두 번째 문자열을 순회하며 Map의 값을 1씩 감소하고 문자가 없거나 값이 0이면 "NO"를 반환
  - 모든 과정을 통과하면 "YES"를 반환
  2.

## 💻 JavaScript 풀이 코드

Solution 1: 단일 Map을 사용하여 첫 번째 문자열로 카운팅하고 두 번째 문자열로 차감하며 검증

```js
function solution1(str1, str2) {
  let answer = "YES";
  let sH = new Map();

  for (const x of str1) {
    if (sH.has(x)) sH.set(x, sH.get(x) + 1);
    else sH.set(x, 1);
  }

  for (const x of str2) {
    if (!sH.has(x) || !sH.get(x)) return "NO";
    sH.set(x, sH.get(x) - 1);
  }

  return answer;
}
```

Solution 2: 문자열을 정렬하여 구조가 완전히 일치하는지 비교

```js
function solution2(str1, str2) {
  const sortedStr1 = str1.split("").sort().join("");
  const sortedStr2 = str2.split("").sort().join("");

  return sortedStr1 === sortedStr2;
}
```
