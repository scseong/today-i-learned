## 연속 부분수열 1

### 1. 문제 정보

- **문제 제목**: 연속 부분수열 1
- **카테고리**: 효율성, 두 포인터

### 2. 문제 설명

- **핵심 목표**: : 수열에서 연속된 숫자들의 합이 정확히 M이 되는 구간의 개수를 구한다.
- **제한 사항**:
  - 수열의 길이 N (최대 100,000)
  - 목표 합 M (최대 1억)
- **입출력 예시**:
  - Input: N=8, M=6, [1, 2, 1, 3, 1, 1, 1, 2]
  - Output: 3

### 3. 접근 방식 및 알고리즘

- **사용한 알고리즘**:
  1. 두 포인터
- **풀이 로직**:
  1. `rt` 포인터로 탐색하며 합을 구하고 합이 m을 초과하면 lt 포인터를 증가
  2. for문을 통해 `rt`를 0부터 N-1까지 순회하며 sum에 값을 누적
  3. while문을 사용하여 sum이 m보다 큰 동안 lt가 가리키는 값을 빼고 lt를 한 칸 이동
  4. sum이 m과 일치하면 answer를 증가

### 4. 성능 측정 결과 (Performance)

| **구분**       | **실행 시간 (평균)** | **시간 복잡도** | **공간 복잡도** |
| -------------- | -------------------- | --------------- | --------------- |
| **Solution 1** | **0.0625ms**         | O(N)            | O(1)            |

## 💻 JavaScript 풀이 코드

Solution 1: rt를 이동하며 sum을 구하고 m을 초과하면 lt를 이동해 sum을 줄이며 m과 일치하는 순간을 카운트

```js
function solution1(m, arr) {
  let answer = 0;
  let sum = 0;
  let lt = 0;

  for (let rt = 0; rt < arr.length; rt++) {
    // 1. rt 포인터를 전진하며 값을 누적
    sum += arr[rt];

    // 2. sum이 목표값 m을 초과하면 lt를 이동시켜 sum을 줄임
    while (sum > m) {
      sum -= arr[lt++];
    }

    // 3. sum이 정확히 m과 일치하면 카운트
    if (sum === m) answer++;
  }

  return answer;
}
```
