## 유효한 팰린드롬

### 1. 문제 정보

- **문제 제목**: 유효한 팰린드롬
- **카테고리**: 문자열 탐색

### 2. 문제 설명

- **핵심 목표**: 알파벳 이외의 문자는 모두 무시하고, 남은 알파벳만으로 회문 여부를 판별
- **제한 사항**: 문자열 길이 100 이하, 대소문자 구분 없음, 알파벳 이외의 문자(숫자, 특수문자, 공백 등)는 무시
- **입출력 예시**:
  - Input: found7, time: study; Yduts; emit, 7Dnuof
  - Output: YES

### 3. 접근 방식 및 알고리즘

- **사용한 알고리즘**:
  1. 정규표현식 필터링 후 비교
- **풀이 로직**:
  1. `toLowerCase()`로 대소문자 통일
  2. `replace(/[^a-z]/g, "")`를 사용하여 영문 소문자가 아닌 모든 문자 제거
  3. 정제된 문자열과 뒤집힌 문자열`split.reverse.join`이 일치하는지 비교

### 4. 성능 측정 결과 (Performance)

| **구분**       | **실행 시간 (평균)**   | **시간 복잡도** | **공간 복잡도** |
| -------------- | ---------------------- | --------------- | --------------- |
| **Solution 1** | **약 0.07ms ~ 0.16ms** | O(N)            | O(N)            |

---

## 💻 JavaScript 풀이 코드

Solution 1: 정규표현식 사용

```js
function solution1(str) {
  const s = str.toLowerCase().replace(/[^a-z]/g, "");

  if (s === s.split("").reverse().join("")) return "YES";
  return "NO";
}
```
