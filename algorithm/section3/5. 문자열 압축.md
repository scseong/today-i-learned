## 문자열 압축

### 1. 문제 정보

- **문제 제목**: 문자열 압축
- **카테고리**: 문자열 탐색, 문자열 가공

### 2. 문제 설명

- **핵심 목표**: 연속되는 문자를 '문자 + 반복횟수' 형태로 압축한다. (단, 횟수가 1이면 생략)
- **제한 사항**: 문자열 길이 100 이하, 모든 문자는 대문자
- **입출력 예시**:
  - Input: KKHSSSSSSSE
  - Output: K2HS7E

### 3. 접근 방식 및 알고리즘

- **사용한 알고리즘**:
  1. 선형 탐색
- **풀이 로직**:
  1. 카운트 변수(cnt)를 1로 초기화하고 문자열의 처음부터 끝까지 순회한다.
  2. 현재 문자(s[i])와 다음 문자(s[i+1])를 비교한다.
  3. 두 문자가 같으면 cnt를 1 증가시키고 다음 루프로 이동한다.
  4. 두 문자가 다르면(문자가 바뀌는 지점) 현재 문자를 결과에 추가한다. 이때 cnt가 1보다 크면 숫자도 함께 추가한다.
  5. 다음 문자를 위해 cnt를 다시 1로 초기화한다.

### 4. 성능 측정 결과 (Performance)

| **구분**       | **실행 시간 (평균)** | **시간 복잡도**                   | **공간 복잡도**                     |
| -------------- | -------------------- | --------------------------------- | ----------------------------------- |
| **Solution 1** | **0.0446ms**         | O(N) (문자열의 길이만큼 1회 순회) | O(N) (결과 문자열 저장을 위한 공간) |

## JavaScript 풀이 코드

Solution 1: 인덱스 직접 비교 방식

```js
function solution1(s) {
  let let answer = "";
  let cnt = 1;

  for (let i = 0; i < s.length; i++) {
    if (s[i] === s[i + 1]) cnt++;
    else {
      answer += cnt > 1 ? s[i] + cnt : s[i];
      cnt = 1;
    }
  }

  return answer;
}
```
