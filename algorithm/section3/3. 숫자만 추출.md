## 숫자만 추출

### 1. 문제 정보

- **문제 제목**: 숫자만 추출
- **카테고리**: 문자열 탐색

### 2. 문제 설명

- **핵심 목표**: 혼합된 문자열에서 숫자만 뽑아내어 유효한 자연수로 변환한다. (앞자리 0 제거 필수)
- **제한 사항**: 문자열 길이 50 이하, 결과 숫자는 100,000,000 이하
- **입출력 예시**:
  - Input: g0en2T0s8eSoft -> 추출: 0, 2, 0, 8 -> 결과: 208

### 3. 접근 방식 및 알고리즘

- **사용한 알고리즘**:
  1. 정규표현식 필터링
  2. 아스키 코드 직접 비교
- **풀이 로직**:
  1. 정규표현식 `/[^0-9]/g`를 통해 숫자가 아닌 문자를 모두 공백으로 치환한 후 + 연산자로 숫자형으로 변환
  2. `charCodeAt()`을 사용하여 각 문자의 아스키 값이 숫자 범위(48~57)에 있는지 확인하고 해당하는 문자만 모아 `parseInt()`로 변환

### 4. 성능 측정 결과 (Performance)

| **구분**       | **실행 시간 (평균)** | **시간 복잡도** | **공간 복잡도** |
| -------------- | -------------------- | --------------- | --------------- |
| **Solution 1** | **0.0622ms**         | O(N)            | O(N)            |
| **Solution 2** | **0.0614ms**         | O(N)            | O(N)            |

## JavaScript 풀이 코드

Solution 1: 반복문을 이용한 직접 비교

```js
function solution1(str) {
  // 숫자가 아닌 모든 문자를 제거한 후 숫자로 변환
  return +str.replace(/[^0-9]/g, "");
}
```

Solution 2: 아스키 코드 활용

```js
function solution2(str) {
  let answer = "";
  for (let s of str) {
    const code = s.charCodeAt();
    // 0(48) ~ 9(57) 사이의 값만 추출
    if (code >= 48 && code <= 57) {
      answer += s;
    }
  }
  return parseInt(answer);
}
```
