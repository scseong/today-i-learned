## 가장 짧은 문자거리

### 1. 문제 정보

- **문제 제목**: 가장 짧은 문자거리
- **카테고리**: 문자열 탐색, 양방향 순회 (Two-pass)

### 2. 문제 설명

- **핵심 목표**: 문자열 내 각 문자가 주어진 타겟 문자 t와 얼마나 떨어져 있는지 최소 거리를 구한다.
- **제한 사항**: 문자열 길이 100 이하, 모든 문자는 소문자
- **입출력 예시**:
  - Input: teachermode e
  - Output: 1 0 1 2 1 0 1 2 2 1 0

### 3. 접근 방식 및 알고리즘

- **사용한 알고리즘**:
  1. 선형 탐색
- **풀이 로직**:
  1. 정방향 순회 (왼쪽 -> 오른쪽): 타겟 문자를 만날 때마다 거리를 0으로 초기화하고 다른 문자를 만나면 거리를 1씩 증가시키며 기록
  2. 역방향 순회 (오른쪽 -> 왼쪽): 동일한 방식으로 거리를 계산하되 이미 기록된 값(정방향 결과)과 비교하여 더 작은 값으로 업데이트

### 4. 성능 측정 결과 (Performance)

| **구분**       | **실행 시간 (평균)** | **시간 복잡도**                 | **공간 복잡도**    |
| -------------- | -------------------- | ------------------------------- | ------------------ |
| **Solution 1** | **0.0833ms**         | O(N) (2번의 전체 순회이므로 2N) | O(N) (answer 배열) |

## JavaScript 풀이 코드

Solution 1: 정방향 탐색과 역방향 탐색 후 각 인덱스의 거리를 비교하여 짧은 값 선택

```js
function solution1(s, t) {
  let answer = Array.from(s).fill(0);

  for (let i = 0, p = 0; i < s.length; i++) {
    if (s[i] === t) p = 0;
    else answer[i] = ++p;
  }

  for (let i = s.length - 1, p = 0; i >= 0; i--) {
    if (s[i] === t) p = 0;
    else answer[i] = Math.min(answer[i], ++p);
  }

  return answer;
}
```
