## K번째 큰 수

### 1. 문제 정보

- **문제 제목**: K번째 큰 수
- **카테고리**: 완전탐색

### 2. 문제 설명

- **핵심 목표**: 카드 $N$장 중 3장을 선택해 더한 모든 합계 중, 중복을 제외하고 $K$번째로 큰 수를 구한다.
- **제한 사항**:
  - 카드 수 $N$ (최대 100), 순위 $K$ (최대 50).
  - $K$번째 수가 없으면 -1을 반환한다.
- **입출력 예시**:
  - Input: 10, 3, 13 15 34 23 45 65 33 11 26 42
  - Output: 143

### 3. 접근 방식 및 알고리즘

- **사용한 알고리즘**:
  1. 3중 조합 탐색
- **풀이 로직**:
  1. 3중 조합으로 합계를 계산
  2. Set 객체인 list에 합계를 추가하여 동일한 합계 값이 중복 저장되는 것을 방지
  3. 배열 변환 후 내림차순 정렬
  4. k-1 인덱스 값이 존재하지 않을 경우를 대비해 ?? -1

## 💻 JavaScript 풀이 코드

Solution 1: 3중 조합으로 합계를 구한 뒤 Set으로 중복을 제거하고, 내림차순 정렬하여 K번째 값을 반환

```js
function solution1(n, k, card) {
  let answer = 0;
  const list = new Set();

  for (let x = 0; x < n - 2; x++) {
    for (let y = x + 1; y < n - 1; y++) {
      for (let z = y + 1; z < n; z++) {
        list.add(card[x] + card[y] + card[z]);
      }
    }
  }

  answer = [...list].sort((a, b) => b - a)[k - 1] ?? -1;
  return answer;
}
```
