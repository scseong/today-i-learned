## 뒤집은 소수

### 1. 문제 정보

- **문제 제목**: 뒤집은 소수
- **카테고리**: 완전탐색

### 2. 문제 설명

- **핵심 목표**: 주어진 자연수들을 뒤집은 뒤, 그 숫자가 소수(Prime Number)인 것들만 찾아 출력한다.
- **제한 사항**:
  - 자연수의 개수 N (3 ≤ N ≤ 100)
  - 뒤집은 후 첫 자리의 0은 무시 (예: 910 -> 19)
  - 각 자연수 크기 ≤ 100,000
- **입출력 예시**:
  - Input: 32 55 62 20 250 370 200 30 100
  - Output: 23 2 73 2 3

### 3. 접근 방식 및 알고리즘

- **사용한 알고리즘**:
  1. 수학적 연산을 통한 자릿수 분리
- **풀이 로직**:
  1. 자릿수 뒤집기: while 루프와 나머지 연산을 통해 숫자를 역순으로 조합한다.
  2. 소수 판별 (isPrime): 1은 예외 처리하고, 2부터 해당 숫자의 제곱근까지 루프를 돌며 나누어 떨어지는지 확인

### 4. 성능 측정 결과 (Performance)

| **구분**       | **실행 시간 (평균)** | **시간 복잡도**                                  | **공간 복잡도** |
| -------------- | -------------------- | ------------------------------------------------ | --------------- |
| **Solution 1** | **0.11ms**           | O(N \* sqrt{M}) (N은 숫자 개수, K는 최대 자릿수) | O(N)            |

## 💻 JavaScript 풀이 코드

Solution 1: 수학적으로 자릿수 추출 & 제곱근까지 나누어떨어지는지 확인

```js
function isPrime(num) {
  if (num === 1) return false;
  for (let i = 2; i <= Math.sqrt(num); i++) {
    if (num % i === 0) return false;
  }
  return true;
}

function solution1(arr) {
  let answer = [];
  let [n, temp] = [0, 0];

  for (let num of arr) {
    temp = num;
    while (temp > 0) {
      n = n * 10 + (temp % 10);
      temp = parseInt(temp / 10);
    }

    if (isPrime(n)) answer.push(n);
    [n, temp] = [0, 0];
  }

  return answer;
}
```
